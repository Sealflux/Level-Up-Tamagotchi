# Level-Up-Tamagotchi
A gamified virtual pet PCB with LED level-up system. Made with Help From "Tamagotchi PCB Tutorial" by @TaniWanKenobi

## Why I Built This

I wanted to learn PCB design and add one of my favorite things in the world to the Tamagotchi tutorial by gamifying it with LED level indicators. This is a common theme in some of my projects, gamifying things.
